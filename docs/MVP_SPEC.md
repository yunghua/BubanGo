# BubanGo MVP Spec

## 產品定位
BubanGo 是給飲料店、小吃店、餐飲店使用的臨時缺班補人平台。

## MVP 目標
讓店家可以快速發布 2-3 小時的缺班需求，讓打工者可以看到附近缺班並申請接班。

## 角色
1. 店家 / 老闆
2. 打工者 / 補班者

## MVP 核心功能

### 店家端
- 註冊 / 登入
- 建立店家資料
- 發布缺班
  - 日期
  - 開始時間
  - 結束時間
  - 時薪
  - 工作地點
  - 工作內容
  - 人數
- 查看申請者
- 接受某位申請者
- 標記完成

### 打工者端
- 註冊 / 登入
- 建立個人資料
  - 姓名
  - 手機
  - 可工作地區
  - 經驗
- 瀏覽缺班
- 申請缺班
- 查看已申請 / 已錄取班次

## MVP 暫不做
- 金流
- 自動媒合
- 評分系統
- 複雜排班
- Threads 串接
- LINE 官方帳號完整整合
- 原生手機 App

## 技術方向
- 先做 Web App / PWA
- 前端：Next.js
- 後端：Supabase
- 資料庫：PostgreSQL
- 登入：Email / 手機 OTP 後續再加
- 通知：MVP 先用畫面內狀態，之後再接 LINE

## MVP 頁面
1. 首頁
2. 登入 / 註冊
3. 店家後台
4. 發布缺班頁
5. 缺班列表頁
6. 缺班詳情頁
7. 打工者個人頁
8. 申請紀錄頁
